# 📱 네이버 플레이스 세션 생성 가이드

## 🎯 개요

네이버 스마트플레이스는 공식 API를 제공하지 않기 때문에, **세션 생성 도구**를 사용하여 로컬에서 로그인한 후 세션을 클라우드로 업로드하는 방식을 사용합니다.

**걱정하지 마세요! 아주 간단합니다!** 🚀

---

## ⏱️ 소요 시간

**최초 설정: 약 3분**
- 도구 다운로드: 10초
- 로그인 및 2단계 인증: 2분
- 세션 업로드: 30초

**세션 갱신 시: 약 2분** (7일마다 1회)

---

## 📋 준비물

- ✅ 네이버 스마트플레이스 계정
- ✅ Windows PC
- ✅ Chrome 브라우저 (자동 설치됨)
- ✅ 휴대폰 (2단계 인증용)

---

## 🚀 Step-by-Step 가이드

### **Step 1: 세션 생성 도구 다운로드** (10초)

1. 웹 앱 로그인
2. 네이버 플레이스 페이지로 이동
3. **"세션 생성 도구 다운로드"** 버튼 클릭

파일명: `NaverSessionCreator.exe` (약 25MB)

---

### **Step 2: 도구 실행** (5초)

1. 다운로드한 `NaverSessionCreator.exe` 파일을 **더블클릭**

⚠️ **Windows Defender 경고가 나타나면:**
   - "추가 정보" 클릭
   - "실행" 버튼 클릭
   - (정상적인 보안 경고입니다. 안전합니다!)

---

### **Step 3: 계정 정보 입력** (20초)

프로그램 화면이 나타나면:

```
┌───────────────────────────────┐
│  🔐 네이버 세션 생성기        │
├───────────────────────────────┤
│  네이버 아이디:               │
│  [yourname@naver.com___]      │  ← 입력
│                               │
│  비밀번호:                    │
│  [****************]           │  ← 입력
│                               │
│  [🚀 로그인 시작하기]        │  ← 클릭
└───────────────────────────────┘
```

---

### **Step 4: 자동 로그인** (30초)

버튼 클릭 후:

1. ✅ Chrome 브라우저가 **자동으로 열립니다**
2. ✅ 네이버 로그인 페이지가 **자동으로 나타납니다**
3. ✅ 아이디/비밀번호가 **자동으로 입력됩니다**
4. ✅ 로그인 버튼이 **자동으로 클릭됩니다**

**이제 잠시만 기다리세요!** 🕐

---

### **Step 5: 2단계 인증 완료** (60초)

브라우저에 2단계 인증 화면이 나타납니다:

#### **Option A: SMS 인증**
```
┌────────────────────────────┐
│  📱 휴대폰으로            │
│     인증번호를 전송했습니다│
│                            │
│  010-****-1234             │
│                            │
│  인증번호: [______]        │  ← 입력
│                            │
│  [    확인    ]            │  ← 클릭
└────────────────────────────┘
```

#### **Option B: 네이버 앱 인증**
```
┌────────────────────────────┐
│  📱 네이버 앱에서         │
│     확인 버튼을 눌러주세요 │
│                            │
│  [대기 중...]             │
└────────────────────────────┘
```

→ 휴대폰의 네이버 앱 열기
→ "로그인 허용" 버튼 클릭

---

2단계 인증 완료 후:

```
┌───────────────────────────────┐
│  🎉 완료!                     │
├───────────────────────────────┤
│  네이버 세션이 생성되었습니다!│
│                               │
│  서버에 업로드 완료           │
│  쿠키 수: 36개                │
│  유효 기간: 약 7일            │
│                               │
│  이제 웹 앱에서 네이버 리뷰를│
│  관리할 수 있습니다!          │
│                               │
│  [🌐 웹 앱 열기]             │
│  [✅ 닫기]                   │
└───────────────────────────────┘
```

**축하합니다! 모든 설정이 완료되었습니다!** 🎉

---

## ❓ 자주 묻는 질문 (FAQ)

### **Q1: 비밀번호가 저장되나요?**
**A:** 아니요! 비밀번호는 로그인 시에만 사용되고 즉시 삭제됩니다. 서버에 저장되지 않습니다.

### **Q2: 안전한가요?**
**A:** 네! 
- 오픈 소스 코드
- HTTPS 암호화 통신
- MongoDB에 안전하게 저장
- 네이버 공식 로그인 사용

### **Q3: Mac에서도 되나요?**
**A:** 현재는 Windows 전용입니다. Mac/Linux 버전은 추후 제공 예정입니다.

### **Q4: Chrome이 없으면?**
**A:** 프로그램이 자동으로 Chrome Driver를 다운로드하여 사용합니다. Chrome 브라우저 설치는 권장되지만 필수는 아닙니다.

### **Q5: 여러 계정을 사용할 수 있나요?**
**A:** 현재는 계정 1개만 지원합니다. 멀티 계정 기능은 추후 업데이트 예정입니다.

### **Q6: 세션이 왜 만료되나요?**
**A:** 네이버의 보안 정책으로 인해 로그인 세션은 일정 기간 후 자동으로 만료됩니다.

### **Q7: 2단계 인증을 꼭 해야 하나요?**
**A:** 네! 네이버 계정 보안을 위해 2단계 인증은 필수입니다.

### **Q8: 프로그램이 바이러스로 감지됩니다**
**A:** 정상적인 오진(False Positive)입니다. Windows Defender에서 "허용" 처리하시면 됩니다.

### **Q9: 로그인이 계속 실패합니다**
**A:** 다음을 확인해주세요:
- 아이디/비밀번호가 정확한지
- 2단계 인증을 제한 시간 내에 완료했는지
- 네이버 계정이 잠기지 않았는지

### **Q10: 도구를 다시 다운로드해야 하나요?**
**A:** 아니요! 한 번 다운로드하면 계속 사용할 수 있습니다. 세션 갱신 시에도 같은 파일을 사용하세요.

## 🛠️ 문제 해결

### **문제 1: "Chrome 드라이버를 찾을 수 없습니다"**

**해결:**
1. 인터넷 연결 확인
2. 방화벽 설정 확인
3. 프로그램 재실행

### **문제 2: "서버에 연결할 수 없습니다"**

**해결:**
1. 인터넷 연결 확인
2. 웹 앱이 작동 중인지 확인
3. VPN 사용 시 해제 후 재시도

### **문제 3: "로그인 시간 초과"**

**해결:**
1. 2단계 인증을 2분 이내에 완료
2. 너무 오래 걸리면 프로그램 재시작
3. 다시 시도

### **문제 4: "업로드 실패"**

**해결:**
1. 프로그램 재실행
2. 웹 앱에서 로그아웃 후 재로그인
3. 그래도 안 되면 문의하기
